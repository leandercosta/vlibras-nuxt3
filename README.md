# VLibras (gov.br) to Nuxt 3

Mais informações sobre o [VLibras](https://vlibras.gov.br/)

### To install

```
npm i vlibras-nuxt3
```

### To Use

```
<template>
  <VLibras/>
  <div>
    <p>Olá, bom dia.</p>
  </div>
</template>

<script>
import VLibras from 'vlibras-nuxt3'
export default {
  components:{VLibras}
}
</script>
```
